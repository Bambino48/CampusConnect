# CampusConnect

## Présentation

CampusConnect est un projet académique professionnel réalisé en HTML, CSS et JavaScript natif.  
L'objectif est de concevoir une plateforme moderne dédiée à la présentation et à l'inscription à des événements académiques.

## Problématique

Comment créer une interface front-end complète, élégante et interactive, capable de présenter des événements universitaires de manière professionnelle tout en démontrant des compétences solides en développement web ?

## Objectifs du projet

- concevoir une interface crédible et moderne
- pratiquer HTML sémantique
- maîtriser CSS avancé et responsive design
- manipuler le DOM en JavaScript
- développer des composants interactifs
- produire un projet portfolio-ready

## Fonctionnalités principales

- page d'accueil immersive
- section événements dynamiques
- galerie visuelle premium
- filtres par catégorie
- recherche d'événements
- tri dynamique
- cartes interactives
- modal de détail
- carousel de témoignages
- FAQ accordéon
- formulaire avec validation JavaScript
- mode clair / sombre
- sauvegarde locale avec localStorage
- bouton retour en haut

## Technologies utilisées

- HTML5
- CSS3
- JavaScript

## Compétences démontrées

- structuration d'une interface web complète
- utilisation de Flexbox et Grid
- design responsive mobile / tablette / desktop
- animations CSS
- gestion d'événements JavaScript
- manipulation dynamique du DOM
- validation de formulaire
- stockage local
- organisation modulaire des fichiers

## Structure du projet

- `index.html` : page d'accueil
- `events.html` : catalogue d'événements
- `contact.html` : inscription
- `css/style.css` : design principal
- `css/animations.css` : animations
- `css/responsive.css` : responsive design
- `js/data.js` : données simulées
- `js/main.js` : logique globale
- `js/events.js` : filtres, tri, favoris
- `js/modal.js` : fenêtre de détail
- `js/carousel.js` : carousel
- `js/form.js` : validation du formulaire
- `js/theme.js` : mode sombre

## Ce que j'ai appris

Ce projet m'a permis de mieux comprendre la structuration d'un site moderne, l'organisation d'un front-end sans framework, la manipulation du DOM, la gestion de l'expérience utilisateur et la création d'une interface cohérente pour un portfolio professionnel.

Ouvrir `index.html` dans un navigateur.
