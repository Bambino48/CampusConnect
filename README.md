# CampusConnect

CampusConnect est un projet académique professionnel réalisé en HTML, CSS et JavaScript natif.

## Objectif
Créer une plateforme de présentation et d'inscription à des événements académiques afin de démontrer des compétences front-end complètes.

## Fonctionnalités
- page d'accueil moderne
- catalogue d'événements dynamique
- filtres par catégorie
- recherche et tri
- modal de détail
- carousel de témoignages
- FAQ interactive
- formulaire avec validation JavaScript
- mode clair / sombre
- sauvegarde locale avec localStorage
- responsive design

## Technologies
- HTML5
- CSS3
- JavaScript

## Structure
- `index.html` : accueil
- `events.html` : liste des événements
- `contact.html` : formulaire d'inscription
- `css/` : styles, animations, responsive
- `js/` : logique interactive

## Compétences démontrées
- HTML sémantique
- mise en page avec Flexbox et Grid
- animations CSS
- manipulation du DOM
- gestion des événements
- validation de formulaire
- stockage local
- UI responsive

## Lancer le projet
Ouvrir `index.html` dans un navigateur.
